{{mission}} LaTiS Server

Usage: {{context-root}}/dataset.suffix?projection&selection&filter


dataset: Name of dataset (see Available Datasets)

suffix: Type of output (see Output Options)

projection: List of variables to return with optional hyperslab (index subset) definitions. Defaults to all. The Dataset Descriptor Structure (DDS) will describe the variables for each dataset. Use the dds suffix to get a dataset's DDS.

selection: Zero or more relative constraints on a variable (e.g. time>=2010-01-01). Each must be preceded by a '&'.

filter: Zero or more filters to be applied to the data (see Filter Options). Each must be preceded by a '&'.
